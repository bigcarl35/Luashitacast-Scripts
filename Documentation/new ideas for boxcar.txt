Maybe a change in subsets? If there's only one entry, no need to make it a group. This would change:

    Subset = {
        [1] = 'Default'
    }

to:
    Subset = 'Default',

and a multi entry to:

    Subset = {
        [1] = 'Healing_Magic_Skill',
